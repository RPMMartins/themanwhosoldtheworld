Website/Server Configuration

This Website is hosted using a Python Django Framework within a personal Linux server using an Apache2 configuration. The physical server itself is running on a personal machine currently located in Lisbon. It uses Ubuntu 18.04 server software running in a virtualBox enviroment, which is remotetly controled via SSH connection. The physical server is also controled via VPN connection. Furthemore the Apache2 configuration only allows for secure HTTPS connection which is certified by Let's Encrypt.

More to Come!

Currently the website doesn't have much to offer, but I don't intend to keep it that way for long. Since the website is built on top of a Django Framework, it allows for a great deal of flexibility for backend development. It also allows to take advantage of the full family of Python libraries for all kind of applications.
